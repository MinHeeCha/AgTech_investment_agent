COMPLETE ARCHITECTURE IMPLEMENTED
================================================================================

✅ STAGE 1: STARTUP DISCOVERY
   → StartupDiscoveryAgent
   Output: StartupProfile

✅ STAGE 2: PARALLEL ANALYSIS (4 Agents Concurrent)
   → TechnologySummaryAgent (Novelty, Defensibility)
   → MarketabilityEvaluationAgent (Market Growth, Feasibility)
   → ImpactEvaluationAgent (Environmental, Agricultural)
   → DataMoatAnalysisAgent (Data Assets, Defensibility)
   Output: 4 Independent Analysis Results

✅ STAGE 3: COMPETITOR COMPARISON
   → CompetitorComparisonAgent
   Output: CompetitorAnalysisResult

✅ STAGE 4: INVESTMENT DECISION
   → InvestmentDecisionAgent
   Output: InvestmentDecision (INVEST/HOLD/PASS)

✅ STAGE 5: REPORT GENERATION
   → ReportGenerationAgent
   Output: FullEvaluationResult + Comprehensive Report

QUICK START
================================================================================

1. Setup Environment:
   $ cd AgTech_investment_agent
   $ python -m venv venv
   $ source venv/bin/activate  # Windows: venv\Scripts\activate
   $ pip install -r requirements.txt

2. Configure:
   $ cp .env.example .env
   # Edit .env with your settings

3. Run Evaluation:
   $ python run_evaluation.py

4. Check Results:
   $ ls outputs/
   # Outputs include:
   # - {name}_evaluation_report.txt (detailed text report)
   # - {name}_evaluation_summary.json (structured data)

5. Run Tests:
   $ pytest tests/ -v --cov=.

================================================================================
USAGE EXAMPLES
================================================================================

EXAMPLE 1: Single Startup Evaluation
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
from app import AgentOrchestrator

orchestrator = AgentOrchestrator()
result = orchestrator.evaluate_startup("TechFarm AI", {
    "founded_year": 2021,
    "headquarters": "San Francisco, CA",
    "stage": "Series A"
})

print(f"Recommendation: {result.investment_decision.recommendation}")
print(f"Score: {result.investment_decision.overall_assessment_score:.2%}")
print(f"Confidence: {result.investment_decision.confidence_score:.2%}")

EXAMPLE 2: Multiple Startups
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
startups = ["Startup1", "Startup2", "Startup3"]
results = orchestrator.evaluate_multiple_startups(startups)

for r in results:
    print(f"{r.startup.name}: {r.investment_decision.recommendation}")

EXAMPLE 3: With Document Retrieval
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
from rag import Retriever, DocumentLoader

retriever = Retriever()
docs = DocumentLoader.load_directory("./startup_docs")
retriever.add_documents(docs)

orchestrator = AgentOrchestrator(retriever)
result = orchestrator.evaluate_startup("Startup Name")
